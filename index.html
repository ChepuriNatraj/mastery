🛠️ Essential Commands

📦 Package Creation

# C++ package
ros2 pkg create my_robot_cpp \
  --build-type ament_cmake \
  --dependencies rclcpp std_msgs

# Python package
ros2 pkg create my_robot_py \
  --build-type ament_python \
  --dependencies rclpy std_msgs

🔨 Build Commands

# Build all packages
colcon build --symlink-install

# Build specific package
colcon build --packages-select my_robot

# Source the workspace
source install/setup.bash

🔍 Introspection

# Topics
ros2 topic list
ros2 topic echo /chatter

# Services
ros2 service list

# Actions
ros2 action list

# Parameters
ros2 param list

# Nodes
ros2 node info /my_node

📊 Visualization

# TF frames
ros2 run tf2_tools view_frames

# Node graph
rqt_graph

# RViz2
ros2 run rviz2 rviz2

# Gazebo
ros2 launch gazebo_ros gazebo.launch.py
💡 Pro Tips for Maximum Progress:

✓ Keep terminal and RViz side by side • ✓ Use tmux for multi-pane workflows • ✓ Source workspace in every new terminal • ✓ Commit working code daily • ✓ Break things intentionally to learn debugging

🎓 Learning Philosophy:

This is deliberate practice, not passive reading. You'll write code, break it, fix it, and repeat. Every module has C++ and Python examples. Every concept has hands-on verification steps. Master one concept fully before moving to the next.
