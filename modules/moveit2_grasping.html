Grasping & Gripper Integration Basics

Build a robust pick-place sequence with pre-grasp alignment, gripper control, and planning-scene attach/detach integration.

Grasp Pipeline

  1. Plan to pre-grasp pose.
  2. Cartesian approach.
  3. Close gripper action.
  4. Attach object to tool in planning scene.
  5. Retreat + transport.
  6. Place + open gripper + detach object.

10-Minute Drill

using GripperCommand = control_msgs::action::GripperCommand;
auto client = rclcpp_action::create_client<GripperCommand>(node, "/gripper_controller/gripper_cmd");

GripperCommand::Goal close_goal;
close_goal.command.position = ____; // 0.0 fully closed
close_goal.command.max_effort = 60.0;
client->async_send_goal(close_goal);

moveit_msgs::msg::AttachedCollisionObject aco;
aco.link_name = "tool0";
aco.object.id = "target_box";
aco.object.operation = moveit_msgs::msg::CollisionObject::ADD;
planning_scene_interface.applyAttachedCollisionObject(aco);

Common Failures

  • Slip after close: adjust effort and finger contact geometry.
  • Planner still collides with held object: missing attach state.
  • Unstable approach: add pre-grasp offset and straighten approach vector.
