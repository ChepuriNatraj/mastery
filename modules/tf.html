TF Transforms

Track coordinate frames for sensors and actuators in a unified tree.

Needs RViz Required

What / Why / When

What

TF (Transform) system manages coordinate frames and transformations between them. It maintains a tree where each frame has exactly one parent. Broadcasters publish transforms, listeners query them. TF2 provides time-based lookups with interpolation and extrapolation. Every robot sensor and link has a frame. Common frames: map, odom, base_link, base_footprint, laser_frame, camera_frame.

Why

Sensor data comes in sensor-local coordinates. Navigation plans in map coordinates. Motors act in robot coordinates. TF automatically transforms between frames, handling rotations and translations. It solves "where is the laser scan point relative to the robot base?" without manual matrix math. SLAM, navigation, and manipulation all depend on correct TF trees.

When

Use TF for any spatial relationship: sensor mounts, robot links, world frames. Broadcast static transforms for fixed sensor mounts. Broadcast dynamic transforms for moving parts (wheels, arms). Always visualize your TF tree with view_frames to debug connectivity. A broken TF tree will silently break navigation and SLAM.

10-Minute Drill

C++ Micro-Task

Broadcast and listen to transforms:

#include "rclcpp/rclcpp.hpp"
#include "tf2_ros/transform_broadcaster.h"
#include "tf2_ros/transform_listener.h"
#include "tf2_ros/buffer.h"
#include "geometry_msgs/msg/transform_stamped.hpp"

class TFDemo : public rclcpp::Node {
public:
  TFDemo() : Node("tf_demo"), tf_buffer_(this->get_clock()), 
             tf_listener_(tf_buffer_) {
    
    // Create transform broadcaster
    tf_broadcaster_ = std::make_shared<tf2_ros::TransformBroadcaster>(this);
    
    timer_ = this->create_wall_timer(
      std::chrono::milliseconds(100),
      std::bind(&TFDemo::broadcast_transform, this));
  }

private:
  void broadcast_transform() {
    geometry_msgs::msg::TransformStamped t;
    
    t.header.stamp = this->get_clock()->now();
    t.header.frame_id = "____";  // Parent frame (e.g., "odom")
    t.child_frame_id = "____";   // Child frame (e.g., "base_link")
    
    // Translation (x, y, z in meters)
    t.transform.translation.x = 1.0;
    t.transform.translation.y = 0.5;
    t.transform.translation.z = 0.0;
    
    // Rotation (quaternion)
    t.transform.rotation.x = 0.0;
    t.transform.rotation.y = 0.0;
    t.transform.rotation.z = 0.0;
    t.transform.rotation.w = ____;  // Unit quaternion
    
    tf_broadcaster_->sendTransform(____);
    
    // Lookup transform
    try {
      auto transform = tf_buffer_.lookupTransform(
        "odom", "base_link",
        tf2::TimePointZero);  // Latest available
      
      RCLCPP_INFO_THROTTLE(this->get_logger(), *this->get_clock(), 1000,
        "Transform: [%.2f, %.2f, %.2f]",
        transform.transform.translation.x,
        transform.transform.translation.y,
        transform.transform.translation.z);
        
    } catch (tf2::TransformException& ex) {
      RCLCPP_WARN(this->get_logger(), "TF lookup failed: %s", ex.what());
    }
  }

  std::shared_ptr<tf2_ros::TransformBroadcaster> tf_broadcaster_;
  tf2_ros::Buffer tf_buffer_;
  tf2_ros::TransformListener tf_listener_;
  rclcpp::TimerBase::SharedPtr timer_;
};

int main(int argc, char **argv) {
  rclcpp::init(argc, argv);
  rclcpp::spin(std::make_shared<TFDemo>());
  rclcpp::shutdown();
  return 0;
}

Python Micro-Task

Static transform and coordinate conversion:

import rclpy
from rclpy.node import Node
from tf2_ros import TransformBroadcaster, TransformListener, Buffer
from tf2_ros.static_transform_broadcaster import StaticTransformBroadcaster
from geometry_msgs.msg import TransformStamped, PointStamped
import tf2_geometry_msgs
from tf_transformations import quaternion_from_euler

class TFStaticDemo(Node):
    def __init__(self):
        super().__init__('tf_static_demo')
        
        # Broadcast static transform (sensor mount)
        self.static_broadcaster = StaticTransformBroadcaster(self)
        self.publish_static_transforms()
        
        # Listen to transforms
        self.tf_buffer = Buffer()
        self.tf_listener = TransformListener(self.tf_buffer, self)
        
        self.timer = self.create_timer(1.0, self.transform_point)
    
    def publish_static_transforms(self):
        static_transform = TransformStamped()
        static_transform.header.stamp = self.get_clock().now().to_msg()
        static_transform.header.frame_id = '____'  # base_link
        static_transform.child_frame_id = '____'   # laser_frame
        
        # Laser mounted 0.1m forward, 0.2m up
        static_transform.transform.translation.x = ____
        static_transform.transform.translation.y = 0.0
        static_transform.transform.translation.z = 0.2
        
        # No rotation
        quat = quaternion_from_euler(0, 0, 0)
        static_transform.transform.rotation.x = quat[0]
        static_transform.transform.rotation.y = quat[1]
        static_transform.transform.rotation.z = quat[2]
        static_transform.transform.rotation.w = quat[3]
        
        self.static_broadcaster.sendTransform(____)
    
    def transform_point(self):
        # Point in laser frame
        point_laser = PointStamped()
        point_laser.header.frame_id = 'laser_frame'
        point_laser.header.stamp = self.get_clock().now().to_msg()
        point_laser.point.x = 2.0
        point_laser.point.y = 0.0
        point_laser.point.z = 0.0
        
        try:
            # Transform to base_link frame
            point_base = self.tf_buffer.transform(
                point_laser, '____', timeout=rclpy.duration.Duration(seconds=1.0))
            
            self.get_logger().info(
                f'Point in base_link: [{point_base.point.x:.2f}, '
                f'{point_base.point.y:.2f}, {point_base.point.z:.2f}]')
                
        except Exception as ex:
            self.get_logger().warn(f'Transform failed: {ex}')

def main(args=None):
    rclpy.init(args=args)
    node = TFStaticDemo()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()

Parameters & Config

Expected TF Tree Structure

map
 └─ odom
     └─ base_footprint
         └─ base_link
             ├─ laser_frame
             ├─ camera_frame
             ├─ imu_frame
             └─ wheel_left_link / wheel_right_link

Explanation:
- map: Global fixed frame (from SLAM or known map)
- odom: Continuous odometry frame (drifts over time)
- base_footprint: Robot projection on ground
- base_link: Robot center
- sensor frames: Individual sensor mounts

Common Failures & Fix Patterns

Failure 1: Transform Not Available

Symptom: tf2::LookupException: "base_link" passed to lookupTransform does not exist

Root Cause: No node broadcasting the requested transform, or frame name typo.

Fix: Check TF tree and verify broadcaster is running:

# Check what transforms are being published
ros2 topic echo /tf_static
ros2 topic echo /tf

# Generate tree diagram
ros2 run tf2_tools view_frames

# Verify exact frame name
ros2 run tf2_ros tf2_monitor  # Lists all available frames

# Common typos: "base_link" vs "base_footprint", "map" vs "Map"

Failure 2: Extrapolation Into the Future

Symptom: tf2::ExtrapolationException: Extrapolation into the future

Root Cause: Requested transform timestamp is newer than latest available transform.

Fix: Use latest available transform or wait for data:

// Option 1: Use latest available (most common)
auto transform = buffer.lookupTransform(
  "target", "source", tf2::TimePointZero);

// Option 2: Lookup with timeout
try {
  auto transform = buffer.lookupTransform(
    "target", "source", tf2::TimePointZero,
    tf2::durationFromSec(1.0));  // Wait up to 1 second
} catch (tf2::TransformException& ex) {
  RCLCPP_ERROR(node->get_logger(), "Transform failed: %s", ex.what());
}

Failure 3: Multiple Parents (Broken Tree)

Symptom: TF_REPEATED_DATA error, navigation fails silently.

Root Cause: Two nodes publishing transforms with same child_frame_id but different parents.

Fix: Each frame must have exactly one parent:

# Identify duplicate publishers
ros2 topic info /tf -v
# Look for multiple publishers

# Common mistake: robot_state_publisher AND manual broadcaster for base_link
# Solution: Remove one or remap frames to avoid collision

# Verify tree has no loops
ros2 run tf2_tools view_frames
# Inspect PDF for any cycles or disconnected components

Failure Injection Drill

Exercise: Handle Missing Transform Gracefully

Create node that handles TF failures without crashing:

def safe_transform_point(self, point, target_frame):
    try:
        # Wait up to 1 second for transform
        transform_timeout = rclpy.duration.Duration(seconds=1.0)
        
        # Check if transform exists
        if not self.tf_buffer.can_transform(
            target_frame, 
            point.header.frame_id,
            point.header.stamp,
            transform_timeout):
            self.get_logger().warn(
                f'Transform from {point.header.frame_id} to {target_frame} not available')
            return None
        
        # Perform transform
        transformed = self.tf_buffer.transform(point, target_frame, transform_timeout)
        return transformed
        
    except tf2_ros.LookupException as ex:
        self.get_logger().error(f'Frame not found: {ex}')
    except tf2_ros.ExtrapolationException as ex:
        self.get_logger().error(f'Extrapolation error: {ex}')
    except Exception as ex:
        self.get_logger().error(f'Transform failed: {ex}')
    
    return None

Detection: Stop broadcasting a required transform and observe graceful degradation.

Recovery: Queue operations until transform becomes available or use last known good transform with staleness check.

Warning:

Static transforms published once at startup may not be received by late-joining nodes. Use StaticTransformBroadcaster which periodically republishes, or ensure nodes start in correct order.
